# Interrupt-Driven Water Tank Automation System

## Overview
STM32F103C8T6-based smart water tank automation system using HC-SR04 ultrasonic sensor and interrupt-driven firmware.

## Features
- Automatic pump ON/OFF control
- Manual and Auto modes
- Overflow prevention
- Dry-run protection
- LED and buzzer indication
- Bare-metal programming
- EXTI interrupt handling

## Hardware Used
- STM32F103C8T6
- HC-SR04 Ultrasonic Sensor
- Relay Module
- Buzzer
- LEDs
- Push Button
- DC Pump

## Software Used
- STM32CubeIDE
- Proteus
- Embedded C
